How It Works

All on how this works are in a list:


  • The App Launches
  • The App Creates a little HTTP server
  • App Sets the Server to http://localhost:8080
  • Server Runs
  • Game Connects to Localhost
  • Game can send Requests like POST or GET requests
  • POST is regularly default and also use (requestBody) variable for defined Body command
  • App will recognise those requests from the server sending back to them
  • The App Would do the commands from the request type
  • The app sends a response
  • Game moves on from the request
Packages

List of built-in packages


  • SVH.HTTP (The Package for HTTP Stuff)
  • SVH.WWIS (Windows Wallpaper Interaction Script)
  • SVH.About (A Package to make an About page for your SVH Project for Debug Purposes to edit config files)
  • SVH.Directory (A Package for Directory Stuff)
  • SVH.Nitrogen (Keyboard Messer)
Rules

Rules for publishing SVH Projects and Plugins


  • Don't Use Directory.CreateDirectory, please use the SVH.Directory.CreateDirectory for safety.
  • When making plugins, please add as much safety as you can. Failing to do this will get your plugin rejected.
  • Ensure your plugin has proper error handling, including logging, to help debug issues.
  • All plugins must be tested thoroughly to ensure compatibility with the Open SVH framework.
  • Do not include any code that intentionally causes damage to the user's system or files.
  • Always provide clear documentation for your plugin's functionality and how to use it.
  • Be transparent about any external libraries or dependencies used in your plugin, and ensure they are properly licensed.
  • Maintain a clean code structure. Minimize unnecessary code and ensure your plugin is optimized for performance.
  • When using HTTP requests, ensure the server and data transfers are secured with proper encryption methods (e.g., HTTPS).
Example C# Code

Hereâ€™s an example of C# code for setting up a basic HTTP Function: (This Has A Post Command to open command prompt and print "Hello World")



using System;
using System.Net;
using System.Text;

namespace OpenSVH
{
    class Program
    {
        public static string OpenSVHDirectoryName = "OpenSVH";
        public static readonly string folderPath = Path.Combine(Environment.GetFolderPath(Environment.SpecialFolder.ApplicationData), OpenSVHDirectoryName);

        static void Main(string[] args)
        {
            StartServer();
        }

        static void StartServer()
        {
            HttpListener listener = new HttpListener();
            listener.Prefixes.Add("http://localhost:8080/");
            listener.Start();
            Console.WriteLine("OpenSVH Server started on http://localhost:8080/");

            while (true)
            {
                HttpListenerContext context = listener.GetContext();
                HandleRequest(context);
            }
        }

        static void HandleRequest(HttpListenerContext context)
        {
            HttpListenerRequest request = context.Request;
            HttpListenerResponse response = context.Response;

            if (request.HttpMethod == "GET")
            {
                Console.WriteLine("Received GET request: " + request.Url);
                SendResponse(response, "Hello from OpenSVH!");
            }
            else if (request.HttpMethod == "POST")
            {
                using (var reader = new System.IO.StreamReader(request.InputStream, request.ContentEncoding))
                {
                    string requestBody = reader.ReadToEnd();
                    Console.WriteLine("Received POST data: " + requestBody);
                    if (requestBody == "openCommandPrompt")
                    {
                        OpenSVH.Plugins.Nitrogen.KeyboardTyper.Search("cmd.exe", 100);
                        OpenSVH.Plugins.Nitrogen.KeyboardTyper.Close();
                        Task.Delay(1000).Wait();
                        OpenSVH.Plugins.Nitrogen.KeyboardTyper.TypeWithEffect("echo Hello World!", 100); // Example command to type in CMD, adjust as needed
                    }
                    SendResponse(response, "Data received successfully.");
                }
            }
            else
            {
                SendResponse(response, "Unsupported request method.", 405);
            }
        }

        static void SendResponse(HttpListenerResponse response, string message, int statusCode = 200)
        {
            byte[] buffer = Encoding.UTF8.GetBytes(message);
            response.StatusCode = statusCode;
            response.ContentLength64 = buffer.Length;
            response.OutputStream.Write(buffer, 0, buffer.Length);
            response.OutputStream.Close();
        }
    }
}
